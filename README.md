# Wayfinder 404 - Error Page Template

Una página de error 404 moderna, minimalista y de alto impacto visual, diseñada para mantener la retención de usuarios incluso cuando se pierde la ruta de navegación. Este proyecto utiliza una arquitectura de estilos **inline** para garantizar una integración inmediata y sin dependencias externas en cualquier proyecto web.

## 🚀 Características

- **Estilos Inline:** Portabilidad total. Copia y pega el archivo en cualquier entorno (Django, React, PHP, HTML estático) sin preocuparte por archivos CSS externos.
- **Diseño Dark Mode:** Estética inspirada en interfaces modernas con contrastes suaves y legibilidad optimizada.
- **Efectos Visuales Premium:** Incluye un sistema de resplandor (Glow Effect) y sombras dinámicas mediante filtros de desenfoque.
- **SEO Ready:** Etiquetas Meta y Open Graph preconfiguradas para una presencia profesional en redes sociales y motores de búsqueda.
- **Optimización de Fuentes:** Integración directa con Google Fonts (Roboto).

## 🛠️ Tecnologías

- **HTML5** (Semántico)
- **CSS3** (Inline styles & Flexbox)
- **Google Fonts API**

## 📦 Instalación y Uso

1. **Clona el repositorio:**
```bash
   git clone [https://github.com/tu-usuario/nombre-del-repo.git](https://github.com/tu-usuario/nombre-del-repo.git)

```

2. **Personalización:**
* Reemplaza `img/logo.png` con tu logo corporativo.
* Reemplaza `img/lord.png` con tu ilustración principal.
* Ajusta los enlaces de la barra de navegación en la sección `<nav>`.


3. **Despliegue:**
Simplemente sube el archivo `index.html` y la carpeta `img/` a tu servidor.

## 🔍 Configuración SEO

La página incluye la etiqueta:

```html
<meta name="robots" content="noindex, follow">

```

Esto asegura que los motores de búsqueda no indexen el error 404 como contenido de valor, protegiendo el ranking SEO de tu dominio principal mientras permiten el rastreo de los enlaces de recuperación.
